To run the program:
Extract the files from the zip file. Once the files are extracted double click on the .QAR file and make sure quartus prime is downloaded in your local computer. 
Open up the files in Quartus prime and also create a restored folder in the same directory as the .QAR fileopen modelsim and go to 
File -> New -> Project and create a new project. click on the add existing file and browse to the restored folder which was in the same directory as the .QAR file
Import all the files that were created by the .QAR file. Once the simulation is running the opcode for each operation is in the order of the table of the lab requirements.
